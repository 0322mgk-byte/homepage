# 돈버는 글쓰기 무료 비밀특강 랜딩 페이지 PRD

## 1. 개요
- **프로젝트 설명**: 돈버는 글쓰기 무료 비밀특강을 소개하고, 이메일/카카오톡 리드 수집을 목표로 하는 원스크롤 랜딩 페이지.
- **목적**: 스토리텔링에 관심 있는 1인 지식창업자와 소상공인을 리드로 확보하고, 신청률을 높인다.
- **기대효과**: 짧은 시간 내 방문자 → 구독자 → 수강생 전환. 광고 캠페인과 연동 시 ROAS 상승 기대.

## 2. 목표/KPI
**목표:**
- 이메일 구독자 500명 확보
- 무료특강 신청자 200명 확보

**KPI:**
- CTA 클릭률 25% 이상
- 폼 제출 전환율 15% 이상
- 평균 체류시간 1분 이상

## 3. 타깃 사용자
- **주요 대상**: 1인 지식창업자, 디지털 노마드, 자영업자, 예비 창업자, 블로거, 콘텐츠 크리에이터
- **문제점**: 글쓰기 지식 부족, 수익화 방법 모름, 마케팅 툴 분산 문제, 독자 확보 어려움
- **니즈**: 쉬운 교육, 실전 적용 가능한 글쓰기 전략, 실질적 매출 연결 방법

## 4. 기능/콘텐츠

### Hero
- **강렬한 헤드라인**: "단 3일 만에 독자를 사로잡고 수익을 만드는 글쓰기 비법"
- **서브 헤드라인**: "평범한 글쓰기가 돈이 되는 순간, 지금 바로 경험하세요"
- **CTA 버튼**: "무료 비밀특강 신청하기"

### Story
- **문제 제기**: "왜 열심히 써도 돈이 안 될까?" → 독자의 공감 유도
- **해결책 제시**: 스토리텔링 기법, 구조화된 글쓰기 전략, 수익화 채널 연결
- **권위 세우기**: 
  - "○○ 도서 출간 저자"
  - "누적 수강생 1,000명+"
  - "평균 수강생 수익 증가율 300%"
- **혜택 강조**: 
  - 무료 특강 참여 시 글쓰기 템플릿 3종 제공
  - 수익화 체크리스트 PDF 증정
  - 1:1 피드백 기회 제공 (선착순 50명)

### CTA
- 상단/중단/하단에 반복 배치
- 버튼 문구 변형:
  - 상단: "무료 비밀특강 신청하기"
  - 중단: "지금 바로 글쓰기 마스터하기"
  - 하단: "놓치지 마세요! 오늘만 신청 가능"

### Form
- **필수 항목**: 이름, 이메일, 휴대폰 번호
- **선택 항목**: 관심 분야 (블로그/뉴스레터/전자책/SNS)
- **카카오톡 알림 수신 동의** 체크박스
- **자동 리디렉션**: 폼 제출 후 → 감사 페이지 (특강 안내 + 추가 자료 다운로드)

### Social Proof
- **수강생 후기**: 실제 수강생 인터뷰 영상 썸네일 (3~5개)
- **성과 스크린샷**: 
  - "첫 달 블로그 수익 50만원 달성"
  - "뉴스레터 구독자 1,000명 돌파"
  - "전자책 판매 100부 달성"
- **언론 보도/협업사 로고** (있을 경우)

## 6. 기술 요구사항

### 프론트엔드
- **프레임워크**: Next.js 14+ (App Router)
- **스타일링**: TailwindCSS
- **컴포넌트**: Shadcn UI
- **폼 관리**: React Hook Form + Zod (유효성 검증)

### 백엔드
- **데이터 저장**: 구글 앱스 스크립트 → 구글 시트 연동
- **API 엔드포인트**: `/api/submit-form` (Next.js API Routes)
- **이메일 자동화**: 폼 제출 시 웰컴 이메일 자동 발송 (선택 사항)

### 배포/호스팅
- **플랫폼**: Vercel
- **CI/CD**: GitHub 연동 자동 배포
- **도메인**: 커스텀 도메인 연결 권장

### SEO/Analytics
- **Google Analytics 4** 설치
- **메타 태그**:
  - OG 태그 (페이스북/인스타그램)
  - Twitter 카드
- **구조화된 데이터**: JSON-LD (Course Schema)
- **사이트맵**: 자동 생성
- **robots.txt**: 설정

### 성능 최적화
- 이미지 최적화 (Next.js Image 컴포넌트)
- 폰트 최적화 (next/font)
- Lighthouse 점수 90+ 목표
